# 贪吃蛇大作战

一个有趣的 H5 贪吃蛇游戏，支持多种游戏模式和玩法。

- **游戏名称**：贪吃蛇大作战
- **当前版本**：v1.4
- **游戏类型**：休闲竞技类 H5 游戏
- **核心玩法**：控制蛇移动，吃掉食物变长，躲避碰撞
- **目标用户**：移动端和电脑端玩家

[在线试玩](https://gerryyang2025.github.io/h5-snake-game/) | [帮助文档](./help.html) | [变更日志](./CHANGELOG.md)

## 游戏特色

- **四种游戏模式**：经典模式、限时挑战、大乱斗、大逃杀
- **经典模式 · 磁铁道具**：场上周期性出现磁铁，拾取后一段时间内扩大磁吸范围、加快吸引，带倒计时与范围圈提示（其他模式不出现）
- **AI 对战**：大乱斗 8 条 AI 蛇，大逃杀 16 条 AI 蛇
- **难度与生命**：简单 / 普通 / 困难 / 专家；可选 1 / 3 / 5 条命（限时模式固定 1 条）
- **食物与连击**：多种食物图标、不同分值；短时连续进食可叠连击加分
- **多端支持**：虚拟摇杆、滑动、陀螺仪与键盘（方向键 / WASD，空格加速）
- **表现与反馈**：粒子爆炸效果（彩色爆炸粒子 + 白色火花粒子）、磁吸效果、Slither 风格圆球蛇身与卡通眼睛、加速时蛇头光晕与放射线、小地图（混战模式可显示 AI）；背景音乐与特效音
- **皮肤系统**：12 种预置蛇皮肤（樱花粉、海洋蓝、森林绿、夕阳橙、薰衣草、珊瑚红、柠檬黄、薄荷青、巧克力、冰川蓝、玫瑰红、翡翠绿）；开局随机分配，皮肤影响蛇身色调、蛇头光晕与排行榜高亮色

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 本地运行
open index.html
# 或使用简单 HTTP 服务器
python -m http.server 8080
```

## 游戏模式

| 模式 | 说明 |
|------|------|
| 经典模式 | 无限时间，传送门，生命系统，磁铁道具 |
| 限时挑战 | 90秒倒计时，连击加分 |
| 大乱斗 | 3分钟与8条AI蛇混战 |
| 大逃杀 | 1分钟猎杀16条AI蛇 |

## 操作方式

| 平台 | 控制方式 |
|------|---------|
| 手机 | 虚拟摇杆 / 屏幕滑动 / 陀螺仪 |
| 电脑 | 方向键/WASD 移动，空格加速 |
