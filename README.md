# 贪吃蛇大作战

一个有趣的 H5 贪吃蛇游戏，支持多种游戏模式和玩法。

[在线试玩](https://gerryyang2025.github.io/h5-snake-game/) | [帮助文档](./help.html)

## 游戏特色

- 🎮 **四种游戏模式**：经典模式、限时挑战、大乱斗、大逃杀
- 🤖 **AI 对战**：大乱斗 8 条 AI蛇，大逃杀 16 条 AI蛇
- 📱 **多端支持**：支持手机虚拟摇杆、重力感应和电脑键盘操作
- 🎵 **音效系统**：背景音乐和游戏特效音

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

```bash
# 本地运行
open index.html
# 或使用简单 HTTP 服务器
python -m http.server 8080
```

## 游戏模式

| 模式 | 说明 |
|------|------|
| 🎮 经典模式 | 无限时间，传送门，生命系统 |
| ⏱️ 限时挑战 | 90秒倒计时，连击加分 |
| ⚔️ 大乱斗 | 3分钟与8条AI蛇混战 |
| 🏃 大逃杀 | 1分钟猎杀16条AI蛇 |

## 操作方式

| 平台 | 控制方式 |
|------|---------|
| 手机 | 虚拟摇杆 / 屏幕滑动 / 陀螺仪 |
| 电脑 | 方向键/WASD 移动，空格加速 |

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- 无外部依赖
