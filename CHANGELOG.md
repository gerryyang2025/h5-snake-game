## v1.0 (2026-03-10)

### 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| 经典模式 | 无限时间，传送门，生命系统 |
| 限时挑战 | 90秒倒计时，连击加分 |
| 大乱斗 | 3分钟与8条AI蛇混战 |
| 大逃杀 | 1分钟猎杀16条AI蛇 |

### 操作方式

- **手机**：虚拟摇杆 / 屏幕滑动 / 陀螺仪
- **电脑**：方向键/WASD 移动，空格加速

### 核心特性

- **难度系统**：简单、普通、困难、专家
- **生命系统**：可选1/3/5条命
- **食物系统**：15种emoji食物，不同分值
- **连击系统**：2秒内连续吃食物，最高10倍加分
- **视觉特效**：粒子爆炸、磁吸效果、加速火焰拖尾
- **音效系统**：背景音乐 + 多种特效音
- **小地图**：显示蛇身和食物位置
