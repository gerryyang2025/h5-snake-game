## v1.4 (2026-03-24)

### 新增

- **玩家蛇皮肤系统**：12 种预置颜色皮肤（樱花粉、海洋蓝、森林绿、夕阳橙、薰衣草、珊瑚红、柠檬黄、薄荷青、巧克力、冰川蓝、玫瑰红、翡翠绿）；游戏开局时随机分配一款皮肤；皮肤影响蛇身整体色调、蛇头光晕颜色及排行榜中玩家名字的高亮色

## v1.3 (2026-03-23)

### 新增

- **粒子爆炸效果**：参考 [responsive-snake](https://github.com/iamchrismiller/responsive-snake) 项目，吃到食物时触发逼真的粒子爆炸效果
  - 爆炸粒子：彩色，向外扩散，大小渐变
  - 火花粒子：白色，快速飞散，带重力效果

## v1.2 (2026-03-22)

### 调整

- **蛇身与蛇头视觉**：改为 Slither.io 风格圆球分段；径向光泽渐变、节段外发光；蛇头略大并带粉色调光晕。
- **眼睛**：眼白 + 瞳孔 + 高光，风格更卡通、易辨认朝向。
- **加速表现**：去掉蛇尾多层火焰拖尾；改为蛇头金橙色径向光晕、脉冲中层光晕、内层高光与放射线，强调加速状态。
- **磁铁食物**：绘制时固定使用 🧲，与普通食物的 emoji 展示方式区分开。

---

## v1.1 (2026-03-22)

### 新增

- **经典模式 · 磁铁道具**：与普通食物分池生成，约每 5～15 秒随机尝试投放一枚；拾取后 **15 秒**内扩大磁吸范围、加快食物被拉向蛇头的速度，顶部显示剩余秒数；地图上显示蓝色磁吸范围圈与蛇头光晕，并播放拾取音效。

### 调整

- **主循环节奏**：正常更新间隔改为 **120ms**，按住加速时为 **100ms**（相对更线性的加速手感）。
- **难度**：各难度档位下的加速倍率参数已随新节奏重调。
- **操控**：加速时转向平滑系数略降，减少高速下甩头过猛。

### 说明

- 磁铁仅在 **经典模式** 出现；限时挑战、大乱斗、大逃杀模式下不生成磁铁，相关 HUD 会隐藏。

## v1.0 (2026-03-10)

### 游戏模式

| 模式 | 说明 |
|------|------|
| 经典模式 | 无限时间，传送门，生命系统 |
| 限时挑战 | 90秒倒计时，连击加分 |
| 大乱斗 | 3分钟与8条AI蛇混战 |
| 大逃杀 | 1分钟猎杀16条AI蛇 |

### 操作方式

- **手机**：虚拟摇杆 / 屏幕滑动 / 陀螺仪
- **电脑**：方向键/WASD 移动，空格加速

### 核心特性

- **难度系统**：简单、普通、困难、专家
- **生命系统**：可选1/3/5条命
- **食物系统**：15种食物图标，不同分值
- **连击系统**：2秒内连续吃食物，最高10倍加分
- **视觉特效**：粒子爆炸、磁吸效果；蛇身为 Slither 风格圆球与光泽；加速时蛇头金橙色光晕；经典模式磁铁激活时显示范围圈与蛇头光晕
- **音效系统**：背景音乐 + 多种特效音
- **小地图**：显示蛇身、食物和AI位置（大乱斗/大逃杀模式显示AI红点）
