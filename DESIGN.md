# 贪吃蛇大作战 - 设计文档

## 游戏概述

- **游戏名称**：贪吃蛇大作战
- **游戏类型**：休闲竞技类 H5 游戏
- **核心玩法**：控制蛇移动，吃掉食物变长，躲避碰撞
- **目标用户**：移动端和电脑端玩家

## v4.5 版本更新内容

### 大逃杀·猎手模式

- **模式介绍**：1分钟限时猎杀！玩家扮演"猎手"角色，追杀16条逃跑的AI蛇
- **AI特性**：
  - AI长度随机（3-15节），增加趣味性
  - 三种AI智商可选：简单/普通/困难
  - AI会尽全力躲避玩家
- **击杀规则**：
  - 玩家头部撞击AI蛇的任何部位 → AI被击杀，AI身体转化为食物
  - AI蛇头部撞击玩家身体 → 玩家失败（AI的"意外反杀"）
  - 双方头部相撞 → 同归于尽，游戏结束
- **无敌时间**：游戏开始时2秒无敌时间，防止开局即结束
- **AI之间**：碰撞不会判定失败，只会互相躲避
- **食物机制**：初始没有随机食物，击杀AI后AI身体化为食物
- **成绩评定**：
  - 击杀 10+ 条：🏆 超神表现
  - 击杀 6-9 条：⭐ 表现出色
  - 击杀 3-5 条：💪 不错哦
  - 击杀 0-2 条：💕 再接再厉
- **小地图**：显示AI位置（红点）

### 大乱斗模式优化

- **自身碰撞**：允许自己撞到自己，不判定失败
- **AI之间碰撞**：碰撞不会判定失败，只会互相躲避

### 无敌时间系统

- 游戏开始时默认2秒无敌
- 失败复活后2秒无敌
- 无敌时间内不判定以下碰撞：
  - 自身碰撞
  - 头对头碰撞
  - AI撞到玩家身体

## v4.0 版本更新内容

### 大逃杀·猎手模式（新增）

- **模式介绍**：1分钟限时猎杀！玩家扮演"猎手"角色，追杀16条逃跑的AI蛇。AI会尽全力躲避玩家，你需要在一分钟内尽可能多地击杀AI！
- **目标**：在1分钟内击杀尽可能多的AI蛇！
- **击杀规则**：
  - 头部判定：玩家头部撞击AI蛇的任何部位 → AI被击杀，AI身体转化为食物
  - 身体判定：AI蛇头部撞击玩家身体 → 玩家失败，游戏立即结束
  - 头对头：双方头部相撞 → 同归于尽，游戏结束
  - 边界与自身：玩家撞墙或撞自身 → 游戏结束
- **食物与成长机制**：
  - 大逃杀猎手模式初始没有随机食物
  - 击杀AI后，AI的身体会散落成若干食物点（每节身体变成一个食物）
  - 玩家吃到食物会增长身体长度
  - 策略提示：身体越长，虽然攻击范围越大，但灵活性越低
- **AI智商选择**：简单/普通/困难
- **界面提示**：
  - 击杀数：屏幕顶部中央显示当前击杀数/总AI数
  - 小地图：右下角小地图显示AI位置（红点）
- **动态倒计时**：右上角显示1分钟倒计时，最后10秒红色闪烁警告

## v3.2 版本更新内容

### 大乱斗模式

- **模式介绍**：3分钟限时多人混战
- **AI对手**：8条AI蛇（ 小青、小红、小白、小黄、小紫、小橙、小粉、小灰）
- **AI行为**：自动寻找食物、追踪玩家、智能躲避边界
- **击杀规则**：主动撞击对方蛇身为胜者，被撞者死亡化为食物
- **实时排名**：右上角显示当前排名，每0.5秒更新
- **AI重生**：被击杀的AI蛇会在1-2秒后重新入场
- **游戏结束**：显示最终排名，统计得分和长度

### 新增AI蛇特性

- 每条AI有独特颜色和名字
- AI可以吃食物变长
- AI头部添加眼睛（跟随方向转动）
- 击杀提示文字显示

### 界面优化

- 实时排名显示在右上角（倒计时下方）
- 排名面板使用半透明黑色背景+粉色边框
- 玩家名称高亮显示（绿色+⭐标记）

## v2.1 版本更新内容

### 难度系统

| 难度 | 普通速度 | 加速速度 | 说明 |
|------|---------|---------|------|
| 简单 | 0.25 | 0.4 | 休闲玩家 |
| 普通 | 0.35 | 0.5 | 默认难度 |
| 困难 | 0.5 | 0.7 | 进阶玩家 |
| 专家 | 0.7 | 1.0 | 高手挑战 |

### 生命系统

- 初始3条命（可选择1/3/5）
- 右上角显示剩余生命
- 普通碰撞：保持蛇的状态继续（有无敌时间）
- 边界碰撞：重置到中心，保持原有长度
- 3条命用完才真正结束

### 边界系统

- 可移动区域：坐标1 到 WORLD_SIZE-2
- 边界判定：碰到边界（坐标0或WORLD_SIZE-1）游戏失败
- 边界重置：失败后重置到中心，保持长度
- 无敌时间：失败后2秒无敌

### 食物系统

- 15种emoji食物
- 分数规则：
  - 🍎🍐🍊 = 10分 (+1长度)
  - 🍋🍇🍓 = 20分 (+2长度)
  - 🍑🍒🥝 = 30分 (+3长度)
  - 🍩🍪🍫 = 50分 (+5长度)
  - 🍰🎂🏆 = 100分 (+10长度)
- 加速时额外+50%分数
- 生成规则：40%蛇头附近，60%世界随机
- 数量：最多25个

### 视觉特效

1. **分数弹出**：吃到食物时显示「+分数」飘字
2. **粒子特效**：吃掉食物时彩色粒子爆炸
3. **磁吸效果**：食物靠近蛇头时被吸引
4. **加速特效**：

   - 头部超大光晕（橙→黄→白）

### 音效系统

- 吃食物音效

- 加速音效

### 游戏结束弹窗

## 操作方式详解

### 1. 虚拟摇杆（推荐手机）

- 位置：屏幕左下角
- 用法：拖动圆形摇杆

### 2. 屏幕滑动

- 位置：游戏区域任意位置
- 用法：手指/鼠标滑动

### 3. 加速按钮

- 位置：屏幕右下角
- 图标：🚀
- 用法：长按加速

### 4. 键盘控制（电脑）

| 按键 | 功能 |
|------|------|
| ↑/W | 向上 |
| ↓/S | 向下 |
| ←/A | 向左 |
| →/D | 向右 |
| 空格 | 加速 |

### 5. 陀螺仪（部分手机）

- 倾斜手机控制方向
- 阈值：倾斜角度 > 15° 触发

## 游戏模式

### 🎮 经典模式

- 无限时间，尽可能吃掉更多食物变得更长
- 支持生命系统，失败后可复活
- 特色：传送门 - 穿过可快速穿梭地图

### ⏱️ 限时挑战

- 90秒倒计时，在限定时间内尽可能拿高分
- 按得分和长度排名结算
- 特色：连击系统 - 连续吃食物获得加分加成

### ⚔️ 大乱斗

- 3分钟多人混战，与8名AI对手同场竞技
- 吃掉他们的蛇身让自己更强壮
- 主动撞击对方蛇身为击杀，被撞者化为食物
- 特色：AI对手、击杀规则、实时排名、3分钟倒计时
